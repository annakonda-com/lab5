###  Детали реализации

#### 1. Симуляция полёта (Тема 1)
- **Законы Ньютона:** Ускорение рассчитывается по формуле `a = F / m`; позиция и скорость обновляются с помощью **интегратора Верле**.
- **Гравитация:** Реализован закон всемирного тяготения с использованием **Map‑Reduce** для вычисления суммарной гравитационной силы, действующей на каждый объект, что обеспечивает эффективность при большом количестве небесных тел.
- **Управление вектором тяги:** Задаётся направление двигателя и степень открытия дросселя (`throttle` от 0 до 1).
- **Изменение массы:** Топливо расходуется с заданной скоростью, масса аппарата динамически обновляется (`mass = dryMass + fuelMass`).

#### 2. Симуляция астрономических наблюдений (Тема 2)
- Задача восстановления трёхмерных координат объекта по двум проекциям на сферическую поверхность (с разных точек наблюдения) решается путём вычисления точки пересечения двух лучей и взятия средней точки между ними.
- Пользовательский интерфейс позволяет:
    - В **режиме ввода** задавать проекции точек щелчками мыши на двух 2D-экранах.
    - В **режиме просмотра** вращать и масштабировать восстановленную 3D-сцену.

#### 3. Система «частицы-пружины» (Тема 7)
- Твёрдое тело представлено системой точечных масс, соединённых нелинейными пружинами. Сила пружины вычисляется по закону Гука.
- Используются два типа жёсткости (`k1`, `k2`) для детализации поведения материала.
- Поддерживается гравитация, затухание (`damping = 0.995`) и возможность закрепления отдельных частиц.

#### 4. Броуновское движение (Тема 10)
- Частицы движутся под действием случайной силы (диффузии), генерируемой линейным конгруэнтным генератором.
- Реализованы **парные столкновения** частиц между собой (с коэффициентом восстановления 1.0) и **абсолютно упругие столкновения со стенками** контейнера.
- Пользователь может добавлять новые частицы кликом левой кнопки мыши во время симуляции.
